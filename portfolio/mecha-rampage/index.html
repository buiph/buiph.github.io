Mecha Rampage

https://github.com/buiph/Mecha-Rampage

A prototype for a mech game.

Created with Unity and written in C#

Destroy enemy towers using powerful weapons in this sandbox environment.

  • Move with WASD.
  • Aim with mouse.
  • Shoot with left mouse button.
  • Cycle through arsenal with the scroll wheel.
  • Interact with F.
  • Discard current weapon with V.

This is a school project which focuses on the movement and vehicle-like control of a mech unit with an intuitive and immersive first person perspective. As well as an inventory management where player can interact with 3 different weapon types, equip them, and see them in action.

This project uses 3D rigidbody physics to handle collision detection and object explosions. To add to that, Unity’s particle system and 3D sound was utilized to further immerse the player with visual and sound effects. It also handles player input using Unity’s New Input System, opening up potential for easy gamepad integration.
